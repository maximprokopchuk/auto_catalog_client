# Auto Catalog Client

Client for https://github.com/maximprokopchuk/auto_catalog_api_server

## Usage

You can change API serverURL in `config/config.ts` file

Install dependencies:

``` bash:
npm install
```

Run dev server:

``` bash:
npm run

```

Build:

``` bash:
npm run build
```

## Tests

Run tests:

``` bash:
npm tests
```

## Linter

Run linter:

``` bash:
npm run lint
```

Autoformat code with prettier:

``` bash:
npm run format
```
